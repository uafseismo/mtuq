Learning ObsPy and MTUQ¶

To avoid reinventing the wheel, MTUQ makes extensive use of ObsPy, a widely-used Python package for seismology. If you’re not familiar with Python or ObsPy, an excellent place to start is the ObsPy tutorial.

Processing a set of traces:

from mtuq import ProcessData

function = ProcessData(filter='lowpass', period=5.)
processed_data = data.map(function)
